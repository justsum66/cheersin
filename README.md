# Cheersin - 你的 AI 派對靈魂伴侶

台灣首個酒類教育 SaaS 平台：派對遊戲 × AI 侍酒師 × 品酒學院

## 功能特色

- 100+ 派對遊戲（真心話大冒險、猜酒、骰子酒局）
- AI 侍酒師（Groq + RAG 智能問答）
- 品酒學院（葡萄酒、威士忌、調酒知識）
- 靈魂酒測（找到你的命定酒款）

## 快速開始

```bash
pnpm install
cp .env.example .env.local
pnpm dev
```

## 技術棧

- Next.js 15 + React 19 + TypeScript 5.7
- Tailwind CSS + Framer Motion
- Supabase + Groq SDK + Pinecone
- Zustand + PayPal + Sentry
