# API 路由索引

依 domain 分組，便於導覽與權限/速率規劃。

| 路徑前綴 | 用途 |
|----------|------|
| `admin/` | 管理後台：knowledge CRUD、usage 查詢 |
| `analytics/` | 分析事件 |
| `chat/` | AI 對話 |
| `debug/` | 除錯用（環境限制） |
| `games/` | 遊戲房：rooms CRUD、join、game-state |
| `health/` | 健康檢查 |
| `notifications/` | 通知 |
| `recommend/` | 酒款推薦 |
| `report/` | 檢舉 |
| `subscription/` | 訂閱與 promo |
| `webhooks/` | 外部 webhook（如 PayPal） |
| `wine-knowledge/` | 酒知識查詢 |
