# /learn 頁面排版 15 項任務（50 專家視角）

依 UX、視覺、無障礙、RWD、轉化率等 50 位專家共識，拆成 15 項可驗證的版面/排版任務。

| # | 專家視角 | 任務 | 狀態 |
|---|----------|------|------|
| L1 | 視覺階層 | Hero 主標與副標字級差 1.25x、副標用 text-white/70 提升可讀 | ✅ |
| L2 | RWD | 主容器 max-w 與 padding 斷點統一（px-3 sm:px-6 md:px-8） | ✅ |
| L3 | 觸控 | 所有可點擊元素 min-h-[48px] 或 min-w-[48px] 滿足 44pt 觸控目標 | ✅ |
| L4 | 間距 | 區塊間 mb-6 md:mb-8、區塊內 space-y-3/4 一致 | ✅ |
| L5 | 進度環 | 進度環與積分/等級區水平對齊、視覺群組 | ✅ |
| L6 | 摺疊區 | 任務/歷程可摺疊、aria-expanded、動畫 duration 0.25s | ✅ |
| L7 | 搜尋 | 搜尋欄 pl-10 留圖標位、清除鈕 min-h/min-w 48、aria-busy | ✅ |
| L8 | 篩選 | 等級 tablist role、鍵盤可切換、選中態 primary 邊框 | ✅ |
| L9 | 課程卡片 | 卡片圓角 rounded-xl、hover 邊框/背景過渡 | ✅ |
| L10 | 快捷入口 | 酒類知識/計劃/徽章等入口卡片間 gap-3、視覺分組 | ✅ |
| L11 | 詞彙表 | 詞彙表入口置於知識區塊內、與酒類知識並列 | ✅ |
| L12 | 空狀態 | 無結果時「找不到…」文案與「清除篩選」按鈕明顯 | ✅ |
| L13 | 焦點 | skip link、focus-visible:ring-2 一致、tab 順序合理 | ✅ |
| L14 | 字體 | 標題 font-display/font-bold、內文 text-sm 輔助說明 | ✅ |
| L15 | CTA | 繼續學習/今日目標區塊對比度足夠、按鈕 primary 辨識度高 | ✅ |

**實作摘要：** 上述 15 項已對應至 `src/app/learn/page.tsx` 既有或本輪微調（Hero 副標、容器、間距、詞彙表入口、無障礙與觸控目標）。
