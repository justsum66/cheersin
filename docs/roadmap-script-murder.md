# 酒局劇本殺 — Drunk Script Murder（報告 85–97）

**專家投票：88% 強烈支持**

## 是什麼

劇本殺 + 喝酒遊戲。每劇本 20–40 分鐘、4–8 人；每人手機收到秘密角色卡與線索；透過對話、投票、喝酒懲罰推進劇情。短劇本（20 分鐘）輕鬆搞笑；長劇本（40 分鐘）懸疑推理；18+ 劇本為情侶專屬、含親密任務。

## MVP 實作清單（基礎）

- [ ] 劇本資料結構：script_id、章節、角色、線索、投票節點、懲罰規則
- [ ] 房間模式：建立「劇本殺房」，綁定 script_id，4–8 人加入後分配角色
- [ ] 前端：劇本殺專屬入口（如 `/script-murder`）、角色卡與線索展示、投票 UI
- [ ] 狀態同步：當前章節、誰已投票、懲罰結果（Supabase Realtime 或 game_states）
- [ ] 至少 1 支短劇本（20 分鐘）上線驗證流程

## 技術依賴

- 現有遊戲房 + Realtime；新資料表 scripts / script_chapters / script_roles
- AI 可後續用於劇本變體與自訂角色名（P3）

## 收入影響

可成為最核心付費內容，預計貢獻訂閱收入 **40–60%**。
