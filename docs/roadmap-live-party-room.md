# 派對直播房 — Live Party Room（報告 69–81）

**專家投票：92% 強烈支持**

## 是什麼

4–12 人透過手機加入同一線上房間；無視訊（可選即時語音 WebRTC）；每人手機為控制器；房主選遊戲、所有人同步題目/任務；內建「乾杯」按鈕（同時按下 → 手機震動 + 碰杯音效）。

## MVP 實作清單（基礎）

- [x] 現有遊戲房 API（Supabase game_rooms、game_room_players、game_states）已可支援多人同房
- [ ] 前端：派對直播房專屬入口（如 `/party-room`）與「乾杯」按鈕 UI
- [ ] Realtime：房主選遊戲後廣播當前 game_id 給全房，所有人同步進入同一遊戲畫面
- [ ] 乾杯：全員同時按下時觸發震動 + 音效（navigator.vibrate + 音效檔）
- [ ] 免費房限 4 人、30 分鐘；付費解鎖 12 人、無限時、18+ 遊戲（與現有訂閱 tier 整合）

## 技術依賴

- Supabase Realtime（已有）、現有 useGameRoom / 房間 API
- 預估核心開發約 2–3 週

## 收入影響

月訂閱轉化率預計提升 **3–5 倍**。
